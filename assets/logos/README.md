# OmniClaw Assets 🔒

This directory contains logos, stickers, and visual assets for OmniClaw.

## Files

- `logo.svg` - Main logo in SVG format
- `logo-dark.svg` - Dark mode logo
- `logo-icon.svg` - Icon version for small sizes
- `sticker-1.svg` - Security scanner sticker
- `sticker-2.svg` - CI/CD security sticker
- `sticker-3.svg` - OmniClaw mascot sticker
- `banner.png` - GitHub banner
- `og-image.png` - Open Graph social image

## Logo Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Primary | #FF6B35 | Main accent |
| Secondary | #1E3A5F | Dark blue |
| Background | #0D1B2A | Dark background |
| Text | #FFFFFF | Light text |
| Success | #00D084 | Success indicators |
| Warning | #FFB800 | Warning indicators |
| Danger | #FF4757 | Error indicators |

## Usage

Use these assets for:
- GitHub repository
- npm package
- Social media
- Presentations
- Merchandise
